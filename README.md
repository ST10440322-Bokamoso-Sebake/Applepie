### Profile Portfolio (`profile-personal/`)
A modern, interactive portfolio website for Bokamoso Sebake, showcasing her journey as the founder of Boka's Yarn Market and Computer Science student.

**Features:**
- Modern purple-themed design with smooth animations
- Dark/Light mode toggle
- Fully responsive design
- Interactive sections (Hero, About, Skills, Projects, Contact)
- Contact form with EmailJS integration
- Downloadable documents (Resume, CV, Cover Letter)

**Tech Stack:** React, Framer Motion, EmailJS

See `profile-personal/README.md` for detailed documentation and deployment instructions.

---

### Library Management System
A comprehensive Library Management System with database integration, GUI interface, and API integration for automatic book data retrieval.

**Features:**
- Database design with SQLite
- GUI interface with Tkinter
- Google Books API & Open Library API integration
- Email/SMS notification system
- Book management, member management, and transaction tracking

**Tech Stack:** Python, Tkinter, SQLite
